## 灯塔是什么

灯塔是一个**每日自动更新**的 A 股投资辅助工具,把两件事放在一个站里:

- **看大势**:每晚新闻联播 + 宏观资金/利率/景气/情绪,告诉你市场是进攻、观望还是防守。
- **选个股**:每个交易日收盘后,用威廉·欧奈尔的 **CAN SLIM** 法则(《笑傲股市》、IBD 体系)给全 A 股打分,筛出当天最值得关注的领涨成长股。

它**不是**荐股软件、不报买卖点、不做实时行情。它给你**判断框架 + 每日信号**,帮你把"该不该买、买什么、为什么"想清楚。

## 每日决策工作流

```
21:45  看大盘视野 ── 灯塔信号是什么色?
         🟢进攻 → 明天可以看选股找机会
         🟡观望 → 多看少动,只盯持仓
         🔴防守 → 不开新仓,执行止损
次日盘前  看每日选股 TOP20,挑符合买点特征的
盘中      用个股体检复核(确认七要素过硬)
每周     看命中回看,校准自己的判断
```

**核心铁律**:大盘(灯塔信号)是总开关。🔴防守时,选股再好也不轻易下手——这是 CAN SLIM 的 M。

## 看懂「灯塔分」(0–100)

每只股票的灯塔分由 CAN SLIM 七要素加权合成:

| 字母 | 含义 | 你该看什么 |
|---|---|---|
| **C** 当季收益 | 盈利是否暴涨 | 季度净利同比 +25% 起为佳 |
| **A** 年度收益 | 赚钱能力稳不稳 | 3 年稳健增长 + ROE 高 |
| **N** 新催化 | 有没有新故事 | 新高 / 政策催化 / 新景气 🚩 |
| **S** 供需 | 筹码紧不紧 | 放量上涨、流通盘适中 |
| **L** 领涨强度 | 是龙头还是跟风 | 阶段涨幅排名靠前(替代 IBD RS Rating) |
| **I** 机构 | 聪明钱在不在买 | 主力净流入、机构持仓 |
| **M** 大盘 | 市场给不给力 | 灯塔信号三色 |

**怎么看分**:
- **≥80**:七要素多数强满足,符合 CAN SLIM 买点特征(重点看)。
- **60–80**:部分满足,有亮点也有短板(挑着看)。
- **<60**:暂不符合,观望。
- 分数会因 **M(大盘)** 变化而整体升降——防守市里高分股也会被压低,这是对的。

## 灯塔信号三色(大盘方向 M)

- 🟢 **进攻**:主要指数多头、资金净流入、政策友好。**选股激进,可按信号介入。**
- 🟡 **观望**:信号矛盾或承压。**多看少动,持仓收紧,不开新仓。**
- 🔴 **防守**:指数破位、资金流出。**暂停买入,该止损止损,保住本金。**

约 3/4 的个股跟随大盘——所以先看色,再看股。

## 命中回看怎么看

- **昨日 TOP 今日表现**:昨天选出的 20 只,今天相对沪深 300 跑赢多少。
- **近 30 日胜率**:这段时间 TOP20 跑赢大盘的比例。
- **累计净值曲线**:TOP20 等权组合 vs 沪深 300 的长期对比。

这是灯塔的"成绩单":不是吹自己多准,而是把历史摆出来让你自己验证 CAN SLIM 在 A 股管不管用。

## 常见问题

**Q:灯塔分会变吗?为什么昨天高分今天低了?**
A:会变。数据每日更新(尤其价格、资金、大盘 M),分自然浮动。M 转防守时整体会被压低。

**Q:为什么没有我关注的那只股?**
A:可能被排除池挡掉(ST、次新、停牌、北交所等),或分数不够进榜。用「个股体检」直接查它。

**Q:数据准吗?**
A:来自 akshare 公开行情/财务接口,每项都标来源。部分 IBD 专有评分(如 RS Rating)用公开指标替代并注明。财报季前后财务字段可能滞后。

**Q:能直接照着买吗?**
A:**不能。** 灯塔给框架和信号,不给买卖点承诺。投资决策和风险你自己负责。
